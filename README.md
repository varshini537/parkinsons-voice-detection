# Parkinson’s Disease Detection from Voice Data

## 📌 Problem Statement

Parkinson’s Disease is a neurodegenerative disorder that affects speech. Early detection can significantly improve treatment outcomes.

## 🎯 Objective

To build a machine learning model that can detect Parkinson’s Disease using voice features.

## 🧠 Approach

* Data preprocessing
* Feature extraction from voice signals
* Training ML models for classification

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Google Colab

## 📈 Expected Outcome

A model that predicts whether a person has Parkinson’s Disease based on voice input.

## 🚧 Project Status

🔄 Currently in development (Model training phase completed)

## 🚀 Future Scope

* Model evaluation & accuracy improvement
* Hyperparameter tuning
* Deployment using Streamlit or Flask
* Real-time voice input testing
